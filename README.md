# estudio-pdf-pro-workspace
Estúdio PDF Pro &amp; AI Suite: Uma SPA serverless no Google Workspace para manipulação avançada de PDFs e Inteligência Artificial. Processamento 100% client-side (navegador) garantindo conformidade total com a LGPD. Inclui OCR, Chat com PDF, edição visual e arquitetura modular (Library/Shell) para escalar com custo zero.
